# Study for Mapping Supplementary Cementitious Materials

**Status:** *Work in Progress*
**Note:** The data and analyses in this repository are incomplete and actively evolving.

---

## Overview

This repository contains the working materials, data, and analysis for the capstone project of the **Yale School of the Environment â€“ Environmental Data Science Certificate Program**.

The study investigates methods for mapping, analyzing, and understanding **Supplementary Cementitious Materials (SCMs)** across different regions.
This is an ongoing project, and the contents will be updated frequently as the research progresses.

Because this is active work, some datasets, notebooks, scripts, and outputs may be partial, experimental, or subject to revision.

---

## Repository Structure

The project is organized into the following folders:

- **01_raw_data/**
  Unprocessed, source datasets gathered for the study.

- **02_processed_data/**
  Cleaned, standardized, or transformed datasets derived from the raw data.

- **03_processing_scripts/**
  Scripts responsible for data cleaning, merging, transformation, and extraction workflows.

- **04_analysis_scripts/**
  Analytical scripts, exploratory work, modeling attempts, and analytical workflows.

- **05_outputs/**
  Generated outputs such as tables, visualizations, intermediate results, and exported analysis products.

- **06_archive/**
  Deprecated, superseded, or historical data and scripts retained for reference.
